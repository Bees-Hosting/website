## 🏷️ À propos
Hébergement gratuit utilisant des outils gratuits pour en venir à ses fins.


## 🚀Outils utilisés

- MyOwnFreeHost (hébergement)
- FreeNom (domaine)
- GoGetSSL (certifcation SSL)
- CloudFlare (DNS)
- Xera (interface client)
- Mailgun (SMTP)
